# LoanApprovalPrediction
Loan Approval Predictor is a machine learning project that predicts whether a loan application will be approved based on applicant details. It applies models like Decision Tree, Random Forest, and Gradient Boosting, compares their performance, and provides insights into key features affecting loan approval.

Jupyter Notebook

Scikit-learn (Decision Tree, Random Forest, Gradient Boosting)
